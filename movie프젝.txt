기능 - 검색, 평점
1. 검색 - (1.키워드 2. 카테고리 3. 전체) - 박재현,용은희,천지민
2. 평점 - (1.평점 추가/수정 2. 평점 삭제 3.평점 조회 ) - 이진천,이규리,홍혜진

각 기능에 대해서는 라이브러리로 만들기로 함
DAO클래스에서 각 기능에 대한 메서드 구현이후
하나의 자바 프로젝트의 Controller클래스에서 들어온 값에 따라 jar파일로 가져온 해당 기능(메서드)를 사용

각자 maven프로젝트에서 main을 만들어 사용하다가 배포 할때 불필요한 코드들 제거한 후 합치기로 함
이후 실행을 여러 해보면서 입력에 대한 예외처리 작업

jdbc.properties의 경우 현재는 자바 프로젝트에 resources에 넣었지만
전의 경우 각 jar파일 내부에 저장하여 DBUtil.java에서 불러오게 하려고 하였으나 경로 확인 불가 및 수정을 
못한다는 문제점으로 인하여  jdbc.properties는 최종 자바프로젝트에 넣게 되었으며 DBUtil의 경우에는
컨트롤러에서 미리 입력해둔 값을 매개변수로 넘겨 Connection 객체를 받아와 사용하게 하였음
